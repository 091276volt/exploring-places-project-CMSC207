# Exploring Places with Voltaire Pascual

## Project Title
Exploring Places – A Personal Travel and Lifestyle Website

## Description
This project is a personal website created to showcase selected travel destinations and provide an introduction to the author’s background, interests, and experiences. The website highlights places that have personal significance, including destinations in the Philippines and the United Arab Emirates. It also presents information about the author's hobbies, career, and personal journey. The goal of the project is to practice the fundamentals of web development while building a meaningful and informative website.

## Features
- Personal introduction section with background information
- Hobbies and interests section with images
- Travel destination pages featuring selected locations
- Image slideshow implemented using JavaScript
- Interactive “Fun Fact” button
- Navigation bar for easy page access
- Organized layout with images, descriptions, and external links

## Technologies Used
- HTML for page structure and content
- CSS for styling and layout design
- JavaScript for interactivity and image slideshow functionality
- Google Fonts for typography
- Font Awesome for icons
